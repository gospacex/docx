# DbxWireJaeger Example

## 项目概述

这是一个展示 dbx 与 Wire 注入和 Jaeger 追踪集成的示例项目。通过 dbx 从 yaml 文件加载 MySQL 配置，使用 Wire 注入原生 `*gorm.DB` 句柄，并在真实的 Jaeger 端点上验证每个 GORM 操作的 span 上报。本项目不修改 hubx 顶层 / dbx / otelx / legoB 任何文件。

## Project Overview

This is an example project that demonstrates dbx with Wire injection and Jaeger tracing integration. It loads MySQL configuration from a yaml file using dbx, injects native `*gorm.DB` handles via Wire, and validates that each GORM operation's span is reported to a real Jaeger endpoint. This project does not modify any files in hubx top-level / dbx / otelx / legoB.

## 主要功能

### 核心功能

- **dbx MySQL 配置加载**: 通过 dbx `MySQLConfig` schema 加载 MySQL 单实例配置
- **Wire 注入 *gorm.DB**: 使用 Wire 注入原生 `*gorm.DB` 句柄
- **OTel TracerProvider 三件套**: TestMain 中显式串联 OTel TracerProvider 的三件套设置
- **5 个 GORM CRUD 用例**: 分别覆盖 Create / Get / List / Update / Delete 操作
- **Jaeger 端到端 span 断言**: 通过轮询 Jaeger API 验证 span 上报
- **Makefile 一键起步**: 启动 legoB 的 mysql + jaeger 服务并运行测试

### Key Features

- **dbx MySQL Configuration Loading**: Load MySQL single-instance configuration via dbx `MySQLConfig` schema
- **Wire Injection of *gorm.DB**: Inject native `*gorm.DB` handles using Wire
- **OTel TracerProvider Three-Step Setup**: Explicitly set up OTel TracerProvider's three-step in TestMain
- **5 GORM CRUD Test Cases**: Cover Create / Get / List / Update / Delete operations respectively
- **Jaeger End-to-End Span Assertion**: Validate span reporting via polling Jaeger API
- **Makefile One-Click Setup**: Start legoB mysql + jaeger services and run tests with one command

## 技术细节

### 1. 配置 (mysql.yaml)

```yaml
mysql:
  host: localhost
  port: 3306
  username: testuser
  password: testpass
  database: testdb
  pool:
    max_open_conns: 20
    max_idle_conns: 5
    conn_max_lifetime: 1800

tracing:
  enabled: true
  service: examples-dbxwirejaeger
  exporter: jaeger
  endpoint: localhost:4318
  protocol: http
  sampler_type: always_on
  sampler_ratio: 1.0
```

### 2. Wire 注入 (wire.go)

```go
//go:build wireinject

package main

type Injector struct {
	DB *DbxProvider
}

func InitializeInjector(cfgPath string) (*Injector, error) {
	wire.Build(
		NewDbxProvider,
		wire.Struct(new(Injector), "*"),
	)
	return nil, nil
}
```

### 3. 测试套件 (main_test.go)

- **TestMain**: 设置 OTel TracerProvider 三件套 + wire 注入 + AutoMigrate
- **requireSpanReported**: 轮询 Jaeger API 验证 span 上报
- **TestDbxWireJaeger_***: 5 个 GORM CRUD 用例
